# Lanceman's BattleTech Campaign Tools

These are a collection of BattleTech campaign management web apps covering Chaos Campaign (Campaign Operations), Chaos Campaign Mercenaries, and BattleTech Aces. All are HTML and JavaScript based, run in your browser, work offline, and are installable as Progressive Web Apps.  

These tools have been authored with the assistance of agentic AI.

Access at: https://x-lanceman-x.github.io/BTechCampaignTools/

## Features

### Aces Companion
* Digitial version of Aces campaign record sheets with automated calculations to make in-between sortie time easier.
* Deployment tracker show you how much PV you've committed to a Sortie
* Pilot Cards that make managing SP easy

### Chaos Campaign Mercenaires Companion
* Track your mercenary unit as it grows from Scale to Scale
* Track SP spending easily with automated SP expenditure
* Fully supports classic and Alpha Strike
* Supports optional rules like No BSP and Jump Tracking

### Chaos Campaign Companion
* Easily calculate your unit's Equipment Rating
* Track both Warchest and Support Points as you spend them
* Quick Warchest to Support Point Conversion

### Across the board
* Journal feature for notes or your unit's story
* Track multiple campaigns and switch between them whenever you want
* Manual data export and import
* Import MekBay CSV files to quickly populate your forces
* Multiple device syncing using GitHub Gists
